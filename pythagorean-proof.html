Pythagorean Theorem: a² + b² = c²

🎛️ 控制面板

a = 3
b = 4
a² + b² = c²
3² + 4² = 5²
9 + 16 = 25 ✓

📖 证明步骤

  1. 构造一个直角三角形，两条直角边分别为 a 和 b，斜边为 c
  2. 以三条边为边长，分别向外作三个正方形，面积分别为 a²、b²、c²
  3. 通过几何割补法可以证明：两个小正方形的面积之和等于大正方形的面积
  4. 即：a² + b² = c²
  5. 这就是勾股定理（毕达哥拉斯定理），在西方称为 Pythagorean Theorem
